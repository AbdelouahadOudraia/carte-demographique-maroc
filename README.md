# Carte interactive des douars du Maroc

## Description
Cette application permet de visualiser de manière interactive les 12 régions et 24 930 douars du Maroc.  
Elle fournit des statistiques démographiques détaillées par région ou douar, telles que la population, la répartition hommes/femmes, l’âge et l’état civil.

## Fonctionnalités principales
- Carte interactive des régions et douars.  
- Affichage des statistiques démographiques au clic.  
- Graphiques interactifs : répartition par âge, sexe, état civil.  
- Légendes et infobulles pour une lecture facile.  
- Interface responsive et moderne.

## Technologies utilisées
- HTML / CSS / JavaScript  
- D3.js (carte interactive)  
- Chart.js (graphique dynamique)
